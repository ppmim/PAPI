1. System¶

PANIC computer system is compound by the Dell PowerEdge R720 12th Generation. It is a 2-socket, 2U server that features the Intel Xeon E5-2600 processor family and supports up to 768GB of DDR3 memory. Dell offers the R720 in various backplane configurations with up to 16 2.5-inch internal hard drives or 8 3.5-inch drives.

The PANIC computer features are:

  • 2 x Intel Xeon E5-2643 (3.3 GHz, QPI 8.0 GT/s, 10M cache, 130W, 4C) Turbo
  • 3.5” chassis with up 8 Hard Drives
  • 8 x 8 GB RDIMM 1600 MHz
  • PERC H710p RAID Controller 1 GB cache
  • 2 x 500 GB SATA 7200 3.5”
  • 6 x 2 TB SAS nearline 6 Gb/s 7.2K rpm 3.5”
